# ms-combat
Microservicio combat

Requerimientos:

- Base de datos Laragon de nombre "db_combat"
- Puerto 8097 libre.
- Para usar los metodos, debe existir al menos 1 usuario en la base de datos "db_user".
- Java 21 en IDE.
- Anotaciones de Lombok habilitadas y plugin Lombok habilitado.
- Spring Boot version 4.0.6
- Postman para probar los metodos.
